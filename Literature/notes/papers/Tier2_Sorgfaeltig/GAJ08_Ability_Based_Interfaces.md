# Improving the Performance of Motor-Impaired Users with Automatically-Generated, Ability-Based Interfaces (2008)

**Quelle:** CHI 2008, ACM  
**DOI:** 10.1145/1357054.1357250

## Kernfrage
Kann man durch automatisch generierte, fähigkeitsbasierte UIs (Ability-Based Interfaces) die Performance motorisch beeinträchtigter Nutzer signifikant verbessern?

## Methode
- Zwei Systeme verglichen: SUPPLE (Präferenz-Modell) vs. SUPPLE++ (direktes Motor-Ability-Modell)
- SUPPLE++: One-time Motor Performance Tests → Ability Modeler → UI-Generierung
- Nutzerstudie mit motorisch beeinträchtigten Teilnehmern (N nicht explizit angegeben)
- Metriken: Task Completion Time, Fehlerrate, Nutzerpräferenz

## Wichtigste Ergebnisse
- SUPPLE++ Nutzer waren **26.4% schneller** als Baseline-Interface
- **73% weniger Fehler** mit ability-based Interface
- Starke Nutzerpräferenz für automatisch generierte Interfaces
- Direktes Ability-Modell (SUPPLE++) > indirektes Präferenz-Modell (SUPPLE)
