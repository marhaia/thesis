How the Pipeline Works

Two-Stage Multi-Output architecture for computational estimation of interactional complexity from GUI screenshots.

Stage 1 — Feature Extraction (Image → Vectors)
Stage 1a
Visual Complexity
v ∈ ℝ⁸
8 image-derived features: Shannon entropy, edge density, feature congestion, subband entropy, layout symmetry, chromatic coherence, visual hierarchy, interactive element density.
Rosenholtz et al. (2007) · Shannon (1948) · Tuch et al. (2009)
Stage 1b
Saliency Features
s ∈ ℝ⁵
UMSI++ deep-learning saliency model predicts where users look. Five summary statistics: dispersion, entropy, coverage, peak count, center bias.
Jiang et al. CHI 2023 · Itti & Koch (2001) · Tatler (2007)
Stage 1c
Search Time Model
Jokinen 2020
Cognitive search time estimated via Jokinen’s adaptive feature guidance model, integrating EMMA eye-movement parameters and ACT-R memory decay.
Jokinen et al. (2020) · Salvucci (2001) · Anderson et al. (2004)
↓
Stage 2 — Cognitive Load Estimation (Vectors → Score)
HCEye Sensitivity
Eye-Tracking Features
h ∈ ℝ⁶
Rule-based sensitivity model derived from HCEye dataset (27 participants, 150 web pages, 3 cognitive load conditions via dual-task paradigm).
Das et al. ETRA 2024
Task Descriptor
Task Context
t ∈ ℝ⁶
Task type, search mode, target specificity, time pressure, and search mode are encoded as a weighted vector that modifies the base score.
Kahneman (1973) · Wickens (2008) · Treisman & Gelade (1980)
User Profile
Big Five Personality
p ∈ ℝ⁵
Optional Big Five personality preset (Costa & McCrae, 1992) applies a small modifier based on neuroticism, conscientiousness, and openness.
Serapio-García et al. (2023) · Taubman-Ben-Ari et al. (2004)
↓
Full Feature Vector
Multi-Output Prediction
x = [v, s, h, t, p] ∈ ℝ³⁰
The concatenated vector is passed to the regression model (HCEye rule-based by default, optional trained Random Forest). Three simultaneous outputs: Cognitive Load Score (0–100), Search Efficiency (0–1), Attention Demand (0–1). A mutual coherence check flags implausible combinations.
Breiman (2001) Random Forest · Borchani et al. (2015) Multi-Output Regression · Rosenholtz et al. (2007) Coherence
Validation
HCEye Behavioral Validation
H1 (fewer fixations under high load): p = 2.55×10⁻²⁸, Cohen’s d = 1.47 (large effect). Model prediction deviation: 0.6% from observed ratio.
Das et al. ETRA 2024
UMSI++ Saliency Validation
Validated against UEyes human fixation ground truth (62 participants, 1,980 UI screenshots). CC = 0.896, SIM = 0.806 — above published SOTA range (CC: 0.40–0.70).
Jiang et al. CHI 2023 · Bylinskii et al. (2018)
